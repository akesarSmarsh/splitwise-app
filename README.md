# SplitWise AI - Smart Expense Tracking

A modern expense tracking and splitting app with AI-powered analytics.

## Features
- Track shared expenses
- Split bills among group members
- AI-powered spending insights
- Monthly analytics and charts
- Settlement suggestions with minimized transactions

## Tech Stack
- Frontend: React + Vite + TailwindCSS
- Backend: Node.js + Express
- Storage: JSON file

## Deploy on Render.com (Free)

### Backend
1. Create Web Service
2. Root Directory: `server`
3. Build: `npm install`
4. Start: `node index.js`

### Frontend
1. Create Static Site
2. Build: `npm install && npm run build`
3. Publish: `dist`
4. Add env: `VITE_API_URL=https://your-backend.onrender.com/api`
